## The Lineup (aka The Meal Plan)

**4 prepped meals + 1 freestyle day.** Protein alternates chicken/pork because we believe in *range*.

All cooking uses **coconut oil** and **coconut aminos** - no soy, no shortcuts, no compromise. The Versace of cooking fats.

| Day | Meal | Protein | Containers |
|-----|------|---------|------------|
| 1 | Coconut Adobo Chicken Stew | Chicken | A1 + B1 |
| 2 | Garlic-Ginger Pork + Coconut Oil Veg | Pork | A2 + B2 |
| 3 | Lemon-Annatto Chicken (Inasal Style) | Chicken | A3 + B3 |
| 4 | Kimchi-Ginger Pork Stew | Pork | A4 + B4 |
| 5 | Eggs + Remaining Veg (fresh cook) | Eggs | No container |

**8 containers total.** 4 per person. Labeled A1-A4 and B1-B4. Organization so clean it could be on the cover of Architectural Digest.

## Deploy This Thing (Vercel)

Takes about 30 seconds. Faster than my verse on "Dolce & Gabbana."

1. Push this repo to GitHub (already done, you're looking at it)
2. Go to [vercel.com/new](https://vercel.com/new)
3. Import the repo
4. Click **Deploy**

No build settings needed. Vercel sees the HTML and just knows what to do. Smart like that.

Or use the CLI:

```bash
npm i -g vercel
cd riffraff-chef-hat
vercel
```

## The Three Tabs

### Order
Full ingredient list with quantities and prices from God's Grace Farm. Budget summary. Interactive checklist so you don't forget the coconut aminos (critical error if you do).

### Cooking
Recipe cards with chef guidance for each meal. Warnings on the pork because *last time it came out tough and we don't do that here.* Quick-reference card at the bottom for the chef.

### Containers
Visual map of all 8 containers. Step-by-step sorting instructions for prep day. The consumption schedule laid out cleaner than a spreadsheet at an accounting firm.
